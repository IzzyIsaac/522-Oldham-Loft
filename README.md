# 522 Oldham Loft

Landing page for the rental at 522 Oldham St, Pigeon Forge, TN.

## Live site

Once GitHub Pages is enabled, the site will be available at:
`https://YOUR-USERNAME.github.io/522-oldham-loft/`

## Files

- `index.html` — the full single-page site (HTML, CSS, JS all inline)
- `WEBP/` — all photos used in the page

## Editing

Open `index.html` in any text editor. Photos referenced as `WEBP/filename.webp`.
